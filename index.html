Skema Integrasi

ESP32 → /api/post_data.php (HTTP POST) mengirim suhu, kelembaban, gas, dan pH. File JSON menyimpan log, dan data bisa dikirim ke ThingSpeak sebagai backup cloud.

Kendali Relay

Status fan & aerator disimpan di control.json. Dashboard mengubah nilai tersebut via update_control.php, dan ESP32 membaca status terbaru melalui get_control.php.

Konfigurasi Praktis
  • Letakkan folder ini di laragon/www/tani-pintar
  • Akses via: http://localhost/tani-pintar
  • Pastikan ekstensi cURL aktif untuk kirim ke ThingSpeak
  • ESP32 & PC harus satu jaringan
Edit API Key di api/post_data.php
